Readme.txt for "Sample MMS Creation" zipfile
############################################

You must have Nokia MMS Java Library 1.1 installed on your computer.

Extract all files into a directory where you can compile and run Java,
using the MMS library.

Compile SampleMMSCreation.java, then run SampleMMSCreation.class

You should get Sample.mms as output.  This file can be e.g. copied to the 
mmsin directory of the Series60 terminal emulator for viewing.
